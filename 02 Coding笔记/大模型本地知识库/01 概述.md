## 大模型本地知识库搭建流程概述

搭建基于大模型的本地知识库通常遵循以下步骤：

1.  **选择大模型方案**：
    *   **本地部署大模型**：通过 `Ollama` 等工具方便地搭建。
    *   **在线大模型 API 接口**：直接调用成熟的在线服务。
> [!NOTE] 考量
    > 如果数据不涉及高度敏感或私密信息，推荐直接使用在线大模型的 API 接口，因为普通个人电脑通常只能搭建小尺寸、性能受限的本地大模型，效果可能大打折扣。
2.  **部署对话软件或知识库系统**：
    安装并部署支持大模型对话或本地知识库功能的软件系统。
3.  **配置与应用**：
    *   在部署的软件中**配置大模型接口**。
    *   **配置知识库**：导入并“投喂”特定的数据文档，使大模型能基于这些本地知识提供服务。
    *   **配置对话应用**：根据需求设置对话界面和功能。
