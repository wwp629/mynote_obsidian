## 准备工作：点亮测试

在正式装入机箱前，先在主板上安装核心硬件进行测试，可以避免安装后出现问题再返工。

1.  将 **CPU**、**内存**、**显卡** 安装到主板上。
2.  连接电源的 **CPU**、**主板** 和 **显卡** 供电线。
3.  接通电源，用螺丝刀短接主板上的 `PWRSW` 针脚启动。
4.  若主板指示灯亮、风扇转动，则说明核心硬件无故障。

> [!NOTE] 测试完成后
> 关闭电源，拆下显卡和所有电源线即可，CPU 和内存无需拆卸。

## 第一步：主板部件安装

1.  **检查机箱铜柱**：确保机箱内主板安装铜柱的数量和位置与主板孔位一致，多则除（防短路），少则补（保稳固）。
2.  **安装 CPU**：将 CPU 安装到主板插槽上。
3.  **安装 M.2 固态硬盘**：将其安装到主板 M.2 插槽，并装回主板自带的散热片（需撕掉导热垫保护膜）。
4.  **安装内存条**：将两条内存安装到主板的 **2、4 卡槽**（或1、3卡槽），以开启双通道。
5.  **安装 CPU 散热器**：
    -   安装散热器底座支架。
    -   在 CPU 表面涂抹导热硅脂（无需抹平）。
    -   将散热器主体固定到支架上，拧紧螺丝。
    -   将散热器风扇扣在散热片上，确保风扇朝向内存条一侧吸风（无骨架面为进风口）。
6.  **连接风扇线**：将散热器风扇的控制线接到主板 `CPU_FAN` 接口。若风扇支持 ARGB，将灯光线接到 `RGB_HEADER` 接口。

## 第二步：机箱内安装与理线

1.  **拆卸机箱侧板**：拆下机箱两侧的盖板。
2.  **安装电源**：将电源安装到机箱下方的电源仓，并从机箱外部拧紧固定螺丝。
3.  **拆卸 PCIe 挡板**：根据显卡厚度，拆除机箱背面对应的 PCIe 挡板。
4.  **主板入箱**：将安装好各项组件的主板放入机箱，对准 IO 接口位置，并上螺丝固定。
5.  **机箱理线**：
    -   **基本原则**：所有线缆尽量从机箱背面穿至正面对应接口，保持正面整洁。
    -   **走线参考**：CPU供电线走顶部，主板和USB线走中部，显卡和机箱跳线走底部。
    -   **连接线缆**：将所有线缆插入主板相应接口。
    > [!WARNING] 注意 RGB 接线
    > 12V 4-Pin 的 RGB 接头通常没有防呆设计，务必确认接头上的箭头或 `+12V` 标识与主板接口的 `12V` 引脚对应，插反可能烧毁设备。

## 第三步：安装显卡与收尾

1.  **安装显卡**：按下主板 PCIe 卡槽的卡扣，将显卡插入卡槽，听到“咔”声后代表安装到位。
2.  **固定显卡**：用螺丝将显卡固定在机箱的 PCIe 挡板位置。
3.  **连接显卡供电**：将电源的 PCIe 供电线插入显卡。
4.  **连接 WiFi 天线**：如果主板支持，将 WiFi 天线旋紧在机箱背面的接口上。

## 第四步：安装系统与配置

1.  **安装系统**：连接显示器和键鼠，开机时按 `F8` 或 `DEL` 键进入 BIOS，设置 U 盘为第一启动项，即可开始安装操作系统。
2.  **安装驱动与软件**：系统安装完成后，联网并安装所有必要的驱动程序和控制软件（如华硕的 AURA SYNC）。
3.  **配置 RGB 灯光**：
    -   通过灯光控制软件设置灯效。
    -   建议设置“关机后关闭灯光”，或在 BIOS 中将主板的节能模式设置为 `S4+S5`（路径参考：高级 -> 内置设备 -> RGB LED灯光）。
4.  **完成组装**：确认所有功能正常后，盖上机箱两侧盖板，完成组装。
