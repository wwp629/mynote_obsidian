在 Word 中，尤其是在处理表格时，可以利用 **域 (Field)** 功能实现类似 Excel 的自动递增编号。核心是使用 `SEQ` (Sequence) 域代码。

## 域代码语法解析

`SEQ` 域的基本语法结构如下：

```
{ SEQ 标识符 [书签] [开关] }
```

-   **SEQ**: 域代码的名称，固定不变。
-   **标识符**: 用户自定义的序列名称（类似变量名），用于区分不同的编号序列。例如 `SEQ 图` 和 `SEQ 表` 会是两个独立的序列。
-   **[书签]**: 可选，用于交叉引用。
-   **[开关]**: 可选，用于控制编号的行为，详见下表。

### 主要开关 (Switches) 说明

| 开关          | 功能描述                                                                                     | 示例                               |
| :------------ | :------------------------------------------------------------------------------------------- | :--------------------------------- |
| `\n`          | **下一个序列号** (Next)。这是默认行为，可省略。                                                | `{ SEQ MyList }`                   |
| `\r n`        | **重置序列号** (Reset)。将序列号重置为指定的数字 `n`。若 `n` 省略，则重置为 0。                  | `{ SEQ MyList \r 1 }` (从1开始)    |
| `\c`          | **重复当前号** (Current)。重复前一个最近的序列号，序号不变。                                     | `{ SEQ MyList \c }`                |
| `\s n`        | **按标题级别重置** (Style)。当遇到指定大纲级别 `n` 的标题样式时，重置序列号。                    | `{ SEQ MyList \s 2 }` (遇到标题2时重置) |
| `\h`          | **隐藏结果** (Hide)。用于交叉引用，自身不显示数字。                                              | `{ SEQ MyList \h }`                |
| `\* Format`   | **指定格式**。用于定义编号的显示样式，如 `\* ROMAN` (罗马数字)、`\* Arabic` (阿拉伯数字) 等。 | `{ SEQ MyList \* ROMAN }` (输出 I, II, III) |

## 基本操作

-   **插入域**: 按下 `Ctrl + F9` 会插入一对大括号 `{}`。这是域代码的容器。
    > [!WARNING]
    > 必须使用 `Ctrl+F9` 快捷键插入域括号，手动输入的 `{}` 无效。

-   **更新域**: 选中域或全文 (`Ctrl+A`) 后，按下 `F9` 来更新域的计算结果，使其显示正确的编号。

-   **切换显示**: 按下 `Alt + F9` 可以在“域代码”和“域结果”之间切换显示。

## 应用实例：创建 “Day X” 序列

以下为在实际使用中创建 `Day 1`, `Day 2`... 序列的示例。

### 1. 开始序列

在需要编号的第一个位置，插入以下域代码。`Day` 是固定文本，`SEQ list` 是自动编号部分。

```
Day{ SEQ list }
```

按下 `F9` 更新后，会显示为 “Day1”。复制整个 `Day{ SEQ list }` 到下一个位置并更新域，会自动变为 “Day2”。

![Exported image](Exported%20image%2020251107224229-2.png)

### 2. 重置序列

如果需要在某个位置将编号重置为 1，使用 `\r` 开关。

```
Day{ SEQ list \r 1 }
```

这会将 `list` 序列的计数器重置，并从 1 开始重新编号。

![Exported image](Exported%20image%2020251107224230-3.png)

### 3. 继续序列

在重置后，若想继续编号，**不能** 再复制带有 `\r 1` 的代码。应使用不带重置开关的基础版本。

```
Day{ SEQ list }
```

![Exported image](Exported%20image%2020251107224236-4.png)

> [!NOTE] 关于 `\* MERGEFORMAT`
> 在编辑或复制域代码时，Word 常常会自动添加 `\* MERGEFORMAT` 开关。这用于在更新时尽量保留格式，通常可以忽略或安全删除，不会影响编号逻辑。
