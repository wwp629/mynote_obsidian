word中插入表格，如果要实现像Excel那样自动编号，可以通过域来实现。  
一、域代码：{ SEQ 标识符 [书签] [开关] }  
SEQ：域代码，相当于系统函数名；  
标示符：自定义标示符，相当于自定义变量；  
[书签]：交叉引用的其他标识书签；  
[开关]：  
\c 重复（即序号不变）前面最接近的序列号。 这对于在页眉或页脚中插入章节号非常有用。
 
\h 隐藏字段结果。 使用它来引用交叉引用中的 Seq 字段，而不打印数字。 例如，你可能想要引用编号的章节，但不打印章节号。 如果还存在格式 (\*) 开关，则 \h 开关不会隐藏字段结果。
 
\n 插入指定项的下一个序列号。 这是默认选项，可以缺省。
 
\r n 将序列号重置为指定的编号 n。 例如， { SEQ 图 \r 3 } 将数字开始编号为 3。n缺省的话，默认将数字开始编号为0。
 
\s 重置“s”后标题级别的序列号。 例如， { SEQ 图 \s 2 } 在标题 2 样式 开始编号。

\*指定编号格式，如1、2、3，Ⅰ、Ⅱ、Ⅲ，①、②、③等，很多格式可选，具体参考“选中域”——鼠标右键——“编辑域”——“选项”查看选择想要的格式。

二、域的使用  
1.插入域，使用ctrl+F9插入{}，必须是使用该快捷键插入，键盘输入的{}无效，插入域后在{}中编辑，{ SEQ list }。也可以通过“插入”——“文档部件”——“域”——“类别”下拉选择编号——“Seq”——“选项”——“SEQ”在SEQ后面输入标示符名称——确定就完成域插入。  
2.插入域后，更新域“F9”，显示隐藏域“Alt+F9”。  
3.复制域到其他位置，F9更新域，即可得到自动编码的内容。  
4.如果1中通过快捷键插入域，编辑保存后，不能实现域功能，请仔细检查域代码中的字符前后空格，少了空格会导致该问题。
 
三、实际使用  
用到的代码有：  
Day{ SEQ list \* MERGEFORMAT}  
Day是标号前的固定的标签，将Day{ SEQ list \* MERGEFORMAT}一起复制到其他位置，Day是纯文本不会变化，域代码实现自动编号。

![Exported image](Exported%20image%2020251107224229-2.png)  

Day{ SEQ list \r1 \* MERGEFORMAT \* MERGEFORMAT }  
r1将编码重置，并从1开始重新编码

![Exported image](Exported%20image%2020251107224230-3.png)

Day{ SEQ list \n \* MERGEFORMAT \* MERGEFORMAT \* MERGEFORMAT }，\n可以缺省  
插入指定项的下一个编码，注意此处不能直接使用第二段的  
Day{ SEQ list \r1 \* MERGEFORMAT \* MERGEFORMAT }，因为这个代码中\r1会直接将编码重置为1.
