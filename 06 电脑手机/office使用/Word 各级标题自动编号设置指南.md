本指南旨在详细说明如何通过 Word 的“多级列表”功能，将自动编号与各级标题样式（标题1、标题2等）进行绑定，实现全自动、规范化的文档标题编号。

## 操作步骤

### 准备工作：打开多级列表设置

1.  在 Word 的“开始”选项卡中，找到“段落”功能区。
2.  点击 **多级列表** 图标（通常在“编号”图标右侧）。
3.  在下拉菜单中，选择 **定义新的多级列表**。

![0 0 0 0 0 do Acrobat 0 Q AA AMI Arial AaBbC AaBbC ...](Exported%20image%2020251107223800-0.png)

> [!TIP] 实时预览
> 为方便在设置时实时看到效果，建议先在文档中输入几行文字，并分别应用上“标题 1”、“标题 2”、“标题 3”样式，然后将光标置于应用了样式的行上再进行以下操作。

### 级别一：设置一级标题 (标题 1)

1.  在“定义新的多级列表”对话框的左上角，点击选择要修改的级别 **`1`**。
2.  在右侧，找到 **将级别链接到样式**，在下拉菜单中选择 **标题 1**。
3.  根据需要修改 **输入编号的格式**。例如，要实现 `1.` 的效果，只需确保输入框中有一个带灰色背景的 `1` 和一个手动输入的 `.` 即可。
4.  （可选）点击 **字体** 按钮，设置编号本身的字体、大小、颜色等。
5.  （建议）展开 **更多 >>**，可以设置编号的对齐位置（0）、缩进（0.74）及编号之后（空格），以保持美观。

![M 01 0 0 J Y 0 N a 0 1 ListNum I S 1 u 074 0 D 0 m...](Exported%20image%2020251107223818-1.png)

### 级别二：设置二级标题 (标题 2)

1.  在左上角，点击选择要修改的级别 **`2`**。
2.  将级别链接到样式 **标题 2**。
3.  **清空** “输入编号的格式”输入框中的所有内容。
4.  在 **包含的级别编号来自** 下拉菜单中，选择 **级别 1**。此时，输入框中会出现一个带灰色背景的 `1`。
5.  在灰色背景的 `1` 后面，**手动** 输入一个分隔符，例如 `.`。
6.  在 **此级别的编号样式** 下拉菜单中，选择 `1, 2, 3, ...`。此时，输入框中会自动追加一个灰色背景的 `1`，格式变为 `1.1`。

### 级别三：设置三级标题 (标题 3)

三级标题的设置方法与二级标题类似，只是需要包含前面两个级别的编号。

1.  在左上角，点击选择要修改的级别 **`3`**。
2.  将级别链接到样式 **标题 3**。
3.  **清空** “输入编号的格式”输入框。
4.  从 **包含的级别编号来自** 中选择 **级别 1**，然后手动输入 `.`。
5.  继续从 **包含的级别编号来自** 中选择 **级别 2**，然后手动输入 `.`。
6.  在 **此级别的编号样式** 中选择 `1, 2, 3, ...`。最终格式应为 `1.1.1`。
7.  更高级别的标题（四级、五级）设置方法依此类推。

![Y 1 3 6 8 9 0 a Q 2 u 2 22 0 0 1 ListNum I S 1 2 2...](Exported%20image%2020251107223850-7.png)

## 关键要点总结

> [!IMPORTANT]
> - **灰色背景的数字**：这些是 Word 自动生成的、会递增的编号，**切勿手动输入**。
> - **手动输入的内容**：级别之间的分隔符（如 `.`、`-` 或空格，根据自己想要的格式选择）需要手动在键盘上敲入。
> - **一致性**：完成设置后，文档中所有标题的编号都应通过应用对应的“标题样式”来自动生成，而不是手动输入数字。
