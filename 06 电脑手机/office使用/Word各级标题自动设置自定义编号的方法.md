一、如上图，“编号”选项右侧的“多级列表”，选择“定义新的多级列表”  
PS：一般选中第三级标题的行进行设置，可以实时看到设置效果
 
二、一级标题编号设置：  
默1-3步默认即可，4、5建议这样设置，保持标题缩进美观。  
PS：“要在库种显示的级别”不要修改，全部默认为级别1

![M 01 0 0 J Y 0 N a 0 1 ListNum I S 1 u 074 0 D 0 m...](Exported%20image%2020251107223818-1.png)  

三、二级标题编号设置：  
1.如上图1、2步，选择2、标题2；  
2.如上图第3步，“输入编号的格式”，清空此处；  
3.“包含的级别编号来自”，选择级别1，此处意思为二级标题自动编号所对应的上一级（一级标题）编号自动生成，效果如下图种；  
4.选择级别1后，出现带灰色背景的数字1，这个1表示的是自动生成的上级标题（一级标题）编号，手动键盘敲入“.”；  
5.“此级别的编号样式”，选择需要的样式，如数字，会自动生成目前设置的二级标题编号。

三、三标题编号设置：  
继续设置三级标题编号  
1.如上图1、2步，选择3、标题3；  
2.如上图第3步，“输入编号的格式”，清空此处；  
3.“包含的级别编号来自”，选择级别1，此处意思为二级标题自动编号所对应的上一级（一级标题）编号自动生成；  
4.选择级别1后，出现带灰色背景的数字1，这个1表示的是自动生成的上级标题（一级标题）编号，手动键盘敲入“.”；  
5.继续重复第3步，“包含的级别编号来自”，选择级别2，自动生成上级标题（二级标题）编号，手动键盘敲入“.”；  
6.“此级别的编号样式”，选择需要的样式，如数字，会自动生成目前设置的三级标题编号。  
三级标题编号设置完成后，点击确认即完成编号设置。4、5级标题编号设置方法类似。
