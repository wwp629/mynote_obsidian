## 解读听力图

听力图通过颜色和符号记录双耳的听力状况。

-   **颜色区分**：
    -   **红色**：代表右耳。
    -   **蓝色**：代表左耳。
-   **符号说明**：
    -   **气导听阈**（通过耳机测试）：
        -   右耳：`O` (掩蔽后: `△`)
        -   左耳：`X` (掩蔽后: `□`)
    -   **骨导听阈**（通过骨导振子测试）：
        -   右耳：`<` (掩蔽后: `[`)
        -   左耳：`>` (掩蔽后: `]`)

## 判断听力损失类型

### 2.1 正常听力

-   **判断标准**：气导、骨导听阈均在 **25dBHL** 以内，且两者差值（气骨导差）小于 **10dBHL**。
    ![12 IE X X ZH 9 09](Exported%20image%2020251107223602-1.jpeg)

### 2.2 传导性听力损失

-   **病因**：声波在外耳、中耳传播受阻（如耵聍堵塞、中耳炎等）。
-   **判断标准**：骨导听阈正常（≤ 25dBHL），气导听阈下降（> 25dBHL）。
-   **干预方式**：可通过药物、手术或佩戴助听器改善。
    ![1201212112 1212 g g g g 0 X X 8 009 09 9](Exported%20image%2020251107223606-2.jpeg)

### 2.3 感音神经性听力损失

-   **病因**：内耳（耳蜗）或听神经发生病变。
-   **判断标准**：气导、骨导听阈均下降（> 25dBHL），且气骨导差小于 **10dBHL**。
-   **干预方式**：此类型听力损失通常是永久性的，无法通过药物或手术治愈，**佩戴助听器是主要的干预和补偿手段**。
    ![1 25 250 500 4k 8k 0 0 0 0 0 0 0 0](Exported%20image%2020251107223611-3.jpeg)

### 2.4 混合性听力损失

-   **病因**：传导通路和感音神经部分同时存在病变。
-   **判断标准**：气导、骨导听阈均下降（> 25dBHL），且气骨导差大于 **10dBHL**。

## 助听器选配原则

### 3.1 通道选择

首先判断听损类型为可佩戴助听器的**传导性**或**混合性**，再根据听力曲线选择。

| 听力曲线类型       | 特征                               | 选配建议                                                     |
| :----------------- | :--------------------------------- | :----------------------------------------------------------- |
| **平坦型**         | 各频率听阈值接近。                 | 只需保证助听器功率足够。                                     |
| **渐降型**         | 听力随频率升高而下降，低频听力较好。 | 建议佩戴**耳背式助听器**，避免因低频能量无法散发而产生的“堵耳效应”。 |
| **上升型**         | 低频听力较差，高频听力较好。       | 建议及早选配助听器，优选**耳背式**。                         |
| **陡降型**         | 低频听力接近正常，高频区急剧下降。 | 建议佩戴开放式、**多通道耳背机**，以保证高频补偿的清晰度和舒适度。 |

> [!NOTE] 关于通道数
> 理论上只有**陡降型**听损对多通道技术（用于精细调节不同频率）有刚需。目前市面主流助听器均为多通道，对于非陡降型听损，入门级多通道（如6通道）即可满足基本需求。

### 3.2 功率选择

根据 WHO (1997) 标准，依据平均听阈（500, 1000, 2000, 4000Hz的均值）来划分听损程度及选择相应功率。
