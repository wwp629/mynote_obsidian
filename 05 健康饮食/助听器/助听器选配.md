![12 IE X X ZH 9 09](Exported%20image%2020251107223602-1.jpeg)  

**2.2****传导性听力损失**  
==骨导听阈在正常值范围内（听损未超过====25dBHL====），气导听阈在正常值范围以外（听损超过====25dBHL====），则听力损失为传导性的，这类听力由于声波在外耳、中耳时被阻挡引起的。声波由于耵聍堵塞、鼓膜受损、中耳积液或者中耳听小骨受损、功能异常等在传播过程中受到影响。==**这种类型的听力损失可通过药物治疗、手术修复或佩戴助听器得到改善**==。==

![1201212112 1212 g g g g 0 X X 8 009 09 9](Exported%20image%2020251107223606-2.jpeg)  

**2.3****神经性听力损失**  
==气导、骨导听阈均超过====25dBHL====，且两者差值在====10dBHL====以内，则听力损失为神经性的，感音神经性听力损失是由于耳蜗、听神经病变或两者均损害引起的。当耳蜗病变时，不能将电脉冲传递到神经，因此大脑接收不到需要的信息。==**神经性听力损失无法通过佩戴助听器改善。**

![1 25 250 500 4k 8k 0 0 0 0 0 0 0 0](Exported%20image%2020251107223611-3.jpeg)  

**2.4****混合性听力损失**  
==气导、骨导听阈均超过====25dBHL====，且两者差值在====10dBHL====以外，则听力损失为混和性，混合性听力损失是传导性和感音神经性听力损失综合的结果。==

**3.****助听器选配原则**  
**3.1****通道选择**  
==首选确定听损类型是传导性或者混合性，其次看听损曲线下降类型选配如下表==

|   |   |   |
|---|---|---|
|==平坦型==|==各频率听阈曲线几乎相等==|==只需要助听器功率足够即可==|
|==渐降型==|==听阈每倍频下降====5~10dB====，低频听阈较好==|==耳内式，耳道式，深耳道式助听器低频能量不能散发，会产生堵耳效应，应该佩戴配戴耳背式助听器==|
|==上升型（倒斜坡型）==|==在高频区的听阈失以====5~10dB====上升==  <br>==低频听阈较差==|==听神经病的患者应及早的配戴助听器，配戴助听器后的有效率大约为====50%====，配戴助听器最好以耳背式作为优选。==|
|==陡降型==|低频区听力接近正常，**到1000Hz以上的频率区陡降**|==最好佩戴开放式==**多通道耳背机**==，不仅满足听得到，听得清，而且不会产生不舒适感==|

==只有陡降型才有必要验配多通道耳机（通道多，高频响应好）。目前市面上专业的助听器均为多通道耳机，因此除非是陡降型，其他类型只需要验配基本多通道的助听器即可（====6====通道即可）==
