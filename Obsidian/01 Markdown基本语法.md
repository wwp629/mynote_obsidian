本文档是 Markdown 在 Obsidian 中常用基础语法的快速参考指南。

## 文本格式化

### 标题

使用井号 `#` 加一个空格来创建标题，支持一到六级。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

> [!NOTE] 标题与标签的区别
> `# 标题`（带空格）用于创建标题结构，而 `#标签`（不带空格）用于创建可供搜索的 [[03 标签|标签]]。

### 文本样式

| 样式       | 语法                   | 示例                       | 快捷键 |
| :--------- | :--------------------- | :------------------------- | :----- |
| 粗体       | `**文本**` 或 `__文本__` | `**粗体**`                   | `Ctrl+B` |
| 斜体       | `*文本*` 或 `_文本_`     | `*斜体*`                     | `Ctrl+I` |
| 删除线     | `~~文本~~`               | `~~删除线~~`                 |        |
| 高亮       | `==文本==`               | `==高亮==`                   |        |
| 粗斜体     | `***文本***`             | `***粗斜体***`               |        |
| 行内代码   | `` `代码` ``             | `` `code` ``                 |        |

## 列表

### 无序与有序列表

-   **无序列表**: 使用 `-`、`*` 或 `+` 加空格创建。
-   **有序列表**: 使用 `数字.` 加空格创建。

```markdown
- 列表项一
- 列表项二

1. 任务一
2. 任务二
```

### 任务列表

使用 `- [ ]` 创建任务列表。在 [[02 编辑与预览|阅读视图]] 或实时预览模式下，复选框可以被交互式点击。

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 按 `Tab` 键可创建子任务
```

## 链接与嵌入

### 内部链接 (Wikilink)

使用双中括号 `[[]]` 是 Obsidian 中最常用、最强大的链接方式。

```markdown
这是指向 [[03 标签]] 笔记的链接。
```

### 外部链接

使用标准 Markdown 语法创建指向外部 URL 的链接。

```markdown
[Obsidian 帮助文档](https://help.obsidian.md)
```

### 嵌入内容

在链接前添加感叹号 `!` 可以将内容直接嵌入到笔记中。这适用于图片、音频、PDF，甚至其他笔记。

```markdown
![[附件/图片.png]]
![[另一篇笔记]]
```

## 代码与注释

### 代码块

-   **行内代码**: 使用一对反引号 `` ` `` 包裹。
-   **代码块**: 使用三对反引号 ` ``` ` 包裹，并可在首行后指定语言以获得语法高亮。

````markdown
```javascript
function hello() {
  console.log("Hello, world!");
}
```
````

### 注释

使用 `%% %%` 包裹的文本将成为注释，仅在源码模式和实时预览的编辑状态下可见。

```markdown
这是一个 %%仅在编辑时可见的%% 注释。
```

## 结构元素

### 水平分割线

在单独一行使用三个或更多的 `---`、`***` 或 `___` 来创建一条水平分割线，用于在视觉上分隔内容。

```markdown
---
```
