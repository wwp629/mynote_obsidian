# Obsidian 图片尺寸调整技巧

## 1. 问题描述
Obsidian 默认插入图片后，图片尺寸可能过大，容易影响笔记的编辑和阅读体验。

## 2. 调整方法
在 Obsidian 中，你可以直接在图片引用语法中添加缩放宽度来调整图片大小。

### 2.1 基础语法
图片引用格式为 `![[图片名称.png]]`。

### 2.2 添加宽度参数
在图片文件路径后加上 `|宽度像素值` 即可。

**示例：**

*   **原始图片（不带宽度参数）**
```markdown
    ![[Pasted image 20251108212233.png]]
```

得到的图片如下：
    ![[Pasted image 20251108212233.png]]

*   **调整尺寸为 300 像素宽**
    在图片路径后添加 `|300` (可根据实际需求修改)，例如：
```markdown
    ![[Pasted image 20251108212233.png|300]]
```
如下图所示：
    ![[Pasted image 20251108212233.png|300]]

## 3. 注意事项

> [!TIP] 推荐手动调整
> 建议避免使用插件或修改 CSS 来设置统一的图片缩放大小。因为不同图片可能需要不同的显示尺寸，手动在插入图片后根据实际需求添加宽度像素值，会更加灵活和方便。
> *   **插件问题**: 插件可能存在兼容性或稳定性问题。
> *   **CSS 问题**: 统一的 CSS 规则无法满足所有图片的个性化尺寸需求。
