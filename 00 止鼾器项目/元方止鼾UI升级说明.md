# 元方止鼾APP UI 升级与功能变更说明

## 1. UI 页面通用修改

### 1.1 ICP备案与公司信息调整
-   **页面范围**：1~32页
-   **修改内容**：
    *   删除页面下方文本信息中的 ICP 备案号及公司信息（保留“隐私政策”、“用户协议”勾选项）。
    *   被删除信息挪至 **135~137页下方**。
    *   ICP备案号“苏ICP备2024131859号-2A”右侧添加“>”按钮，跳转至工信部 ICP 备案查询网址：`http://beian.miit.gov.cn/`

### 1.2 购物商城功能删除
-   **页面范围**：66~68页, 100页, 101页, 138页
-   **修改内容**：彻底删除“购物商城”功能及相关页面。
> [!NOTE] 原因
    > 该功能大概率后续不会上线。

### 1.3 页面不使用或占位说明
-   **43页**：此页面不使用。
-   **37, 41, 44, 45, 47页**：涉及动图，先用图片占位，后续动图渲染完成后更换。
-   **135页**：该页面暂时不使用。
-   **APP Logo文件**：提供止鼾器logo、止鼾器logo白底。

## 2. 注册与个人信息相关功能

### 2.1 新增“OSA严重程度”选项
-   **页面范围**：19~32页 (注册页面)
-   **修改内容**：
    *   在年龄与身高栏目中间新增一栏：“OSA严重程度”。
    *   提供下拉可选值：“原发性鼾症（AHI<5）”、“轻度OSA（5≤AHI<15）”、“中度OSA（15≤AHI<30）”、“重度OSA（AHI≥30）”。
    *   此修改由 UI 负责升级完成。
-   **关联页面**：
    *   新增 151、152 页用于显示下拉弹框选择“OSA严重程度”。
    *   69~97页（我的-个人信息修改）新增上述“OSA严重程度”栏目，下拉弹窗格式复用 151、152 页。

### 2.2 后台接口修改 (与OSA严重程度相关)
-   **注册/修改用户信息接口**：
    *   “注册阶段修改用户信息”及“使用阶段修改用户信息”接口需修改，增加一个 `OSA` 字段。
    *   ![[5723a78ca257b45af28bcc53533f52c.png]]
-   **获取用户基本信息接口**：
    *   “获取用户基本信息”接口需对应增加上述数据字段 `OSA`。

### 2.3 APP 前端治疗逻辑调整
-   **修改逻辑**：通过“获取用户基本信息”接口获取用户基本信息，并基于 `OSA` 字段判断治疗限制。
-   **原版本规则**：
    *   ① 普通用户（role非test）每天只能治疗一次，且治疗间隔必须30分钟以上。
    *   ② 用户role = test，解除当日治疗限制，解除当日治疗间隔30分钟限制。
-   **新增规则**：在此基础上，如果 `OSA = 2` 或 `3`，用户每天可以最多治疗两次，且治疗间隔必须30分钟以上。否则同原版本规则 ① 处理。
-   **其他**：其他逻辑不变。

## 3. 特定页面功能与UI调整

### 3.1 初始化按钮位置
-   **页面范围**：33~46页, 49~52页, 54~56页
-   **修改内容**：初始化按钮挪到右上角。

### 3.2 日历与治疗详情
-   **63页**：日期选中18号，底部显示治疗详情为 UI 错误，设计时按原来版本逻辑处理。
-   **64页**：日历筛选功能不做，按原来实现方案，左右滑动（含上方左右键）能滑动月份即可，无需实现此页面的日历筛选功能。

### 3.3 BMI 数值显示
-   **页面范围**：69~71页, 76页, 81~97页
-   **修改内容**：BMI UI 中显示的数值是 UI 随便输入的，不用管，按前面版本实现即可。

### 3.4 客服沟通页面
-   **126页**：页面底部的“人工客服”可点击，点击跳转至 127 页。
-   **127页**：最上方标题“我的目标”错误，修改为“客服沟通”。此页即为“我的”-“客服沟通”点击进来的页面。

### 3.5 主机信息与电极管理
-   **134页** (主机信息)：
    *   蓝牙ID去掉，保留 SN、嵌入式软件版本。
    *   电极、主机图片有占位，等后续产品定型会放渲染图上来。
-   **136、137页**：这两页背景图中 app 图标用错了，应使用 135 页背景图（这两页是由 135 页点击弹窗出来的内容）。
-   **140页**：
    *   两个按钮“更换电极”、“去购买”中，去掉右边的“去购买”。
    *   只保留“更换电极”作为按钮。
> [!NOTE] 原始设计
    > 这里在原版本中就是这样设计的，UI 没有改过来。
