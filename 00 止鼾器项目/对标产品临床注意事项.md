    # Friedman口腔评分介绍如下：
    
    ### 1、Friedman 口腔评分的组成部分
    
    Friedman评分系统主要包括以下三个关键要素：
    
    1. **Friedman舌头位置（FTP）**
        - **定义**：FTP评估舌头在口腔内的自然位置，观察其与软腭、悬雍垂、扁桃体/咽侧柱和硬腭的关系。
        - **评分标准**（分为5级）：
            - **FTP I**：可完整看到悬雍垂和扁桃体/咽侧柱。
            - **FTP IIa**：可看到大部分悬雍垂，但扁桃体/咽侧柱被部分遮挡。
            - **FTP IIb**：可看到软腭全貌至悬雍垂基底部，但无更多结构可见。
            - **FTP III**：可看到部分软腭，但悬雍垂远端不可见。
            - **FTP IV**：仅能看到硬腭，软腭及以下结构完全不可见。
        - **检查方法**：患者张口，不伸舌头，医生重复检查5次，取最一致的观察结果。
        - **意义**：FTP等级越高，表明舌头位置越高，咽部空间越狭窄，气道阻塞风险越大。
    2. **扁桃体大小（Tonsil Size）**
        - **评分标准**（0-4级）：
            - **0级**：扁桃体已切除或完全不可见。
            - **1级**：扁桃体隐藏在咽侧柱内。
            - **2级**：扁桃体延伸至咽侧柱边缘。
            - **3级**：扁桃体超出咽侧柱但未达中线。
            - **4级**：扁桃体延伸至中线，几乎接触对侧。
        - **意义**：扁桃体越大，越可能压迫咽部空间，增加OSAHS的风险。
    3. **体重指数（BMI）**
        - **定义**：BMI是体重（kg）除以身高（m）的平方，用于评估肥胖程度。
        - **分级参考**：
            - BMI < 40：正常或轻度肥胖。
            - BMI ≥ 40：重度肥胖。
        - **意义**：肥胖与上呼吸道脂肪沉积相关，BMI高者气道塌陷风险增加。
    
    ---
    
    ### 2、Friedman分级系统（Friedman Staging System）
    
    基于上述三项指标，Friedman将OSAHS患者分为四个阶段（Stage I-IV），用于预测手术效果和指导治疗：
    
    - **Stage I**：
        - FTP I、IIa或IIb；
        - 扁桃体3或4级；
        - BMI < 40。
        - **特点**：咽部阻塞较轻，手术（如UPPP）成功率高（约80%）。
    - **Stage II**：
        - FTP I、IIa或IIb，扁桃体0、1或2级；
        - 或 FTP III或IV，扁桃体3或4级；
        - BMI < 40。
        - **特点**：阻塞程度中等，手术成功率下降（约38%）。
    - **Stage III**：
        - FTP III或IV；
        - 扁桃体0、1或2级；
        - BMI < 40。
        - **特点**：舌根部阻塞显著，手术成功率低（约8%）。
    - **Stage IV**：
        - BMI ≥ 40；
        - 或伴有明显颅面畸形（如小下颌、中面部发育不良）。
        - **特点**：重度阻塞，多为全身性问题，手术效果极差。
    
    ---
    
    ### 3、临床意义
    
    1. **OSAHS筛查**  
        Friedman评分简单易行，可在初次体检中使用，帮助识别高风险患者。例如，FTP III或IV通常提示OSAHS可能性极高。
    2. **手术成功率预测**  
        该系统最初用于评估UPPP手术的成功率。研究表明，Stage I患者手术效果最佳，而Stage III和IV患者可能需要多水平手术或非手术治疗（如CPAP）。
    3. **治疗规划**  
        通过评估舌根、扁桃体和肥胖程度，医生可判断阻塞的主要部位，决定是否需要舌根手术、扁桃体切除或减重干预。
    4. **与AHI的相关性**  
        多项研究表明，Friedman分级与呼吸暂停-低通气指数（AHI）有一定相关性，分级越高，OSAHS越严重。
